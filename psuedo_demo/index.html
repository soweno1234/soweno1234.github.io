Lets dive deeper into CSS and how it works

If we think of CSS in terms of selectors, properties and values, then a typical CSS entry would look like this

            
                selector {property: value;}

To style the above, we need to add two tags together. We call this CSS Combinators

What are Pseudo Classes?

Again, if we think of CSS in terms of selectors, properties and values, then a psuedo class is an add-on to a selector.

            selector:pseudo_class {property: value;}

We use them for Links

If we want our link to look different, we can simply add the following

            a {color: red;}

But if we want each of our link states to look different then we will use a pseudo class like the following:

  • :hover
  • :active
  • :visited

But we can also make a new link look different
